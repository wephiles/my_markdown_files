#### 2.2.3.2 编写代码

注意，文件名和类名要一样，否则会编译错误。

```java
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, world!");
  }
}
```

![Clip_2024-07-12_20-20-15](./assets/Clip_2024-07-12_20-20-15.png)
